Module Init (.ml)

module Init: sig .. end
A simple and efficient init implementation.
val full_signal_set : int list
Simply an int list which holds every possible signal. None is provided by default in Sys, so one was simply implemented as a list of all possible signals.
val log : ?status:bool -> string -> unit
Pretty-print a message. Serves no practical use. 
val checkpid : unit -> unit
Check the acquired PID is indeed 1, and suspend execution if not.
val spawn : string -> string array -> unit
Spawn a forked process for the given command/filename, create a new session for it and unblock all signals.
val reap_children : unit -> unit
Wait until all child processes have exited. Recurs until an exception is raised indicating all children have been reaped.
val wait_forever : unit -> unit
Pause and wait for a signal using sigsuspend, then recur.
val install_handler : int -> (unit -> unit) -> unit
Install signal handler function for the given signal.
